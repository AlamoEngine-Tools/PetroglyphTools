    Parameters
    Type Name Description
    System.String relativeFilePath

    The relative file path as defined in the *.MEG file.
    Usually this file path is relative to the game or mod's DATA directory, e.g. Data/My/file.xml when the containing meg file resides in c:/My/Game/Data
    System.Int32 offset

    Optional offset from the start of the *.MEG file archive. If not set, this offset is -1. If a *.MEG file is read from disc using the standard implementation of the IMegFileProcessService this is the byte offset from the start of the file.
    System.UInt32 size

    Optional file size in byte. If not set, this is 0. If a *.MEG file is read from disc using the standard implementation of the IMegFileProcessService this is set to the size in bytes of the packaged file.
