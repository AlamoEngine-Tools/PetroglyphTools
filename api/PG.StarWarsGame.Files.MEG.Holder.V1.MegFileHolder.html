    Class MegFileHolder

    The provided holder for Petroglyph's v1 *.MEG files. *.MEG or Mega files are a proprietary archive type bundling files together in a RAM friendly way. This file holder does not hold all files that are packaged in a *.MEG file, but all necessary meta-information to extract a given file on-demand.
