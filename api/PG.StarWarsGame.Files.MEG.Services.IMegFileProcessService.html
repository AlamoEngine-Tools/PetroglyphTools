    Interface IMegFileProcessService

    MEF service interface definition for handling sorted *.DAT files. A default implementation is provided in MegFileProcessService. When requesting the default implementation via an IoC Container or registering via injection, you may pass a file system as argument implementing and a logger factory implementing
